Política de privacidad (Dolar+)

Esta política describe cómo se maneja la información en relación con la aplicación Dolar+.

Información recopilada

Dolar+ no recopila información personal identificable del usuario.

Uso de información

La aplicación puede consumir datos públicos de APIs de cotización para mostrar tipos de cambio.

Servicios de terceros

La aplicación no utiliza SDKs de terceros para seguimiento o analítica publicitaria.
