Política de Privacidad (Dolar+)

Esta política describe cómo se maneja la información en relación con la aplicación Dolar+.

Información recopilada

La aplicación utiliza herramientas de analítica de terceros (como Mixpanel) para recopilar datos de uso anónimos con el fin de mejorar el producto.

Estos datos pueden incluir eventos de interacción dentro de la aplicación, tipo de dispositivo, versión del sistema operativo y métricas técnicas.

La aplicación no recopila información personal identificable, datos de cuentas ni datos financieros del usuario.

Uso de la información

Los datos de analítica se utilizan exclusivamente para comprender el uso de funcionalidades, mejorar el rendimiento y optimizar la experiencia del usuario.

Las cotizaciones mostradas en la aplicación provienen de APIs públicas y se utilizan únicamente para mostrar información de mercado dentro de la app.

Servicios de terceros

La aplicación utiliza servicios de analítica de terceros (como Mixpanel). Estos servicios pueden recopilar identificadores técnicos del dispositivo (por ejemplo, IDFV) y dirección IP con fines estrictamente analíticos.

Los datos de analítica no están vinculados a la identidad del usuario y no se utilizan con fines publicitarios.

La aplicación no utiliza SDKs de publicidad ni realiza seguimiento entre aplicaciones o sitios web con fines publicitarios.

Almacenamiento de datos

Los datos ingresados por el usuario en el Ledger se almacenan exclusivamente de forma local en el dispositivo y no se transmiten a servidores externos.
