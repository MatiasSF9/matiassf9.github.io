Política de privacidad

Esta política describe cómo se recopila, utiliza y protege la información en relación con las aplicaciones publicadas.

Información recopilada

Dolar+ no recopila información personal identificable del usuario.

Uso de información

La aplicación puede consumir datos públicos de APIs de cotización.

Servicios de terceros

No se utilizan SDKs de terceros para seguimiento o analítica.

Retención de datos

No se almacenan datos personales en servidores externos.

Derechos del usuario

El usuario puede contactar para cualquier consulta relacionada con privacidad.
